## 黃金價格預測小工具

本工具可根據「地緣政治風險」、「美元強度」、「通膨預期」三個因素模擬黃金價格，適合金融分析、投資決策與教學用途。

### 🛠 技術堆疊
- Python
- Streamlit
- Matplotlib

### 📈 模擬邏輯
- 地緣政治風險 ↑ → 黃金 ↑
- 美元走弱（強度 ↓）→ 黃金 ↑
- 通膨預期 ↑ → 黃金 ↑
